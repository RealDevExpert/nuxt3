# @intlify/nuxt3

Nuxt 3 module for vue-i18n-next

> ⚠️ NOTE: This is a WIP module for Nuxt 3, It isn't available yet.

## 🌟 Features
- Setup `vue-i18n` for your Nuxt project

## 🚀 Usage
Add `@intlify/nuxt3` module to `buildModules` options of `nuxt.confg.[ts|js]`

```js
export default {
  buildModules: [
    ['@intlify/nuxt3']
  ]
}
```
